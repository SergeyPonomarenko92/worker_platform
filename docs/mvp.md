# MVP — Worker Platform

## Цель
Веб‑приложение (Украина, UI/контент украинской) для малого бизнеса и мастеров, которое объединяет:
- портфолио/посты + «истории» как в Instagram
- поиск/предложение услуг и товаров как в OLX
- отзывы и рейтинг только после сделки как в Rozetka

## Роли
- **Клієнт**: шукає послугу/товар, переглядає профілі, ініціює заявку/угоду.
- **Провайдер (бізнес/майстер)**: веде профіль, публікує портфоліо, історії, пропозиції (послуги/товари).

## Потоки (MVP)
1) Реєстрація/логін (Breeze)
2) Профіль провайдера (бізнес‑сторінка)
3) Пропозиції: послуги та товари
4) Портфоліо (пости/роботи)
5) Історії (24h)
6) Угода (мінімальна модель «сделка») → після завершення клієнт може залишити відгук

## Антифрод для відгуків (мінімум)
- Відгук можна залишити **лише по завершеній угоді**.
- 1 відгук від клієнта на 1 угоду.

## Перші екрани
- Landing / Welcome
- Catalog (пошук/фільтри)
- Provider profile (хедер + історії + останні роботи)
- Offer details
- Create deal / request
- Leave review
